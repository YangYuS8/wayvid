# Retired AUR Packaging Reference

This directory is kept as a historical reference for the old Arch Linux AUR packaging work.

The legacy `wayvid` and `wayvid-git` AUR packages are not the current installation path for the actively maintained product.

Do not treat the files in this directory as current user installation instructions or an active release pipeline.

If legacy packaging needs to be inspected, review the `PKGBUILD` files here as archived examples only.
