# Legacy Wayvid Summary

The retired `wayvid` framing centered on the previous application story and roadmap. That framing is no longer the basis for the repository after the reset to a Linux dynamic wallpaper platform focused on Wallpaper Engine migration users.

We keep technical knowledge selectively as retained assets and migration candidates, such as playback, library, packaging, and desktop integration lessons. We do not keep the old product framing, release narrative, or implied scope as the default direction for future work.
