# OpenSpec Reset Baseline Archive Note

As part of the repository reset, the legacy OpenSpec current-state specs and active change directories for the retired `wayvid` product story were removed from the live baseline.

Historical change records were preserved under `openspec/changes/archive/` using `legacy-reset-*` names so the reset can start from a clean OpenSpec current state without losing prior proposal history.

The reset-era product direction now lives in `openspec/config.yaml` and the product documentation under `docs/product/`.
