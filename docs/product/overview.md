# Product Overview

## Mission

Build a Linux dynamic wallpaper platform that gives Wallpaper Engine migration users a polished desktop application for discovering, importing, understanding, and running wallpapers on Linux.

## First-Release Focus

- Workshop-centered discovery and acquisition orchestration
- compatibility visibility as a user-facing feature
- `video` and `scene` runtime support
- library-first daily use
- Chinese and English user-facing support

## Why This Reset Exists

The previous `wayvid` framing was too narrow for the product now being defined. The repository reset keeps technical knowledge that still matters, but it no longer treats the old application story as the foundation for the next release.

## Non-Goals for the First Release

- full `web` runtime parity
- creator tools
- cloud or community systems
- advanced automation rules
